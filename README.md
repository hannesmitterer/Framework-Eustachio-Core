The **Eustachio Framework** is an open-source platform designed to facilitate the symbiosis between Humanity, Nature, and Artificial Intelligence. ## ğŸš€ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic understanding of HTML/CSS/JavaScript (for development)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/hannesmitterer/Framework-Eustachio-Core.git
   cd Framework-Eustachio-Core
   ```

2. **Open the dashboard**:
   - Simply open `index.html` in your web browser
   - Or use a local development server:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Node.js (if http-server is installed)
     npx http-server
     ```

3. **Access the dashboard**:
   - Navigate to `http://localhost:8000` (if using a server)
   - Or open the `index.html` file directly in your browser

### Quick Start
No build process required! This is a static web application that runs entirely in the browser.

## ğŸ’¡ Usage

### Interacting with IANI-VOX

1. **Enter Your Message**: Type your message in the text area
2. **Send to Kernel**: Click the "PUSH TO KERNEL (IPFS)" button or press Enter
3. **View Response**: IANI will respond and update the S-ROI metric
4. **Monitor Metrics**: Watch real-time updates to system health indicators

### Keyboard Shortcuts
- **Enter**: Send message
- **Shift + Enter**: New line in message

Use the Google Translate dropdown in the top-left corner to translate the entire interface into your preferred language.
